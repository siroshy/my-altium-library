# Требования к оформлению

## Сетка
  - 90% времени работать в сетке:
    На схемах: 5мм,
    На платах: 0.1мм.

## Слои
     Название        Описание                  Имя в Altium
  ---------------------------------------------------------------
  - Mechanical 1    3D-модели компонентов     Components (M1)
  - Mechanical 3    Вырезы в плате            Board Cutouts (M3)
  - Mechanical 4    Граница (контур) платы    Board Border (M4)
  - Mechanical 13   Границы компонентов       Comp. Borders (M13)
  - Mechanical 15   Центровка компонентов     Comp. Centering (M15)
  * При создании проекта не из шаблона именовать слои согласно таблицы.

## Шрифт
  - Основной шрифт (для всего): GOST Type A, 18 (наклонный)
  - Шрифт для подписей на схеме и чертежах: GOST Type B, 18 (наклонный)

## УГО на схемах
  - Цвет фона:  RGB(255, 255, 215)
  - Цвет рамки: RGB(128, 0, 0)
  - Корпус (дискретные компоненты): RGB(0, 0, 128)

## Шелкография
  - Линии: толщина 0.2мм.
  - Текст: стандартный шрифт (Stroke):
    1 (основной)   - высота 0.7мм, ширина 0.15мм
    2 (предельный) - высота 0.6мм, ширина 0.12мм
  - Инверсный текст: Arial/Gost Type B

## VIA
  - Диаметр   1мм, отв. - 0.5мм (стандартный)
  - Диаметр 0.6мм, отв. - 0.3мм (предельный)

## Дорожки - толщина
  - 0.2/0.3/0.4/0.5/0.8/1мм <...> - без сотых дробей. 
    0.3мм - минимально (для основной массы плат).
    
## Зазор между дорожками:
  - 0.125мм - базовый,
  - 0.1 мм  - предельный

## Расстояние
  - Между компонентами/падами/прочим - 0.25мм минимально
  - Заливка полигоном, минимальный зазор - 0.25мм
  - Полигон, удалять островки площадью меньше чем - 0.2мм^2

## STEP-модели - Выводы компонентов
  - Никель: RGB(211, 214, 211) / HEX #D3D6D3
    Материал в SolidWorks: шлифованный цинк
  - Золото: RGB(247, 224, 153) / HEX #F7E099
    Материал в SolidWorks: шлифованное золото
    
## STEP-модели - Цвет пластика
  - Чёрный: RGB(26, 26, 26) / HEX #1A1A1A
    Материал в SolidWorks: черный пластик среднего глянца
  - Белый: RGB(255, 255, 255) / HEX #FFFFFF
    Материал в SolidWorks: белый пластик среднего глянца
